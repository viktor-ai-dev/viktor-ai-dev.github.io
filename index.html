🔍 3. Hybrid Search (Vector + Keyword)

Combines semantic embeddings with keyword matching for precise information retrieval.

vector_docs = retriever.invoke(rewritten_question)
keyword_docs = [
    doc for doc in vector_docs
    if any(word in doc.page_content.lower() for word in rewritten_question.lower().split())
]
unique_docs = []
seen = set()
for doc in vector_docs + keyword_docs:
    if doc.page_content not in seen:
        unique_docs.append(doc)
        seen.add(doc.page_content)
top_docs = sorted(unique_docs, key=lambda d: score(d), reverse=True)[:5]

Value to Employers: Shows expertise in combining modern retrieval methods.
